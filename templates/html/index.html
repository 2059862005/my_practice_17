现代化的用户认证系统

基于 Django REST Framework 和 JWT 实现安全可靠的用户注册、登录和认证功能。

API 端点

🔐 认证接口

POST /api/auth/login/邮箱密码登录
POST /api/auth/login_with_code/验证码登录
POST /api/auth/register/用户注册（需验证码）
POST /api/auth/logout/用户登出
POST /api/auth/send_code/发送验证码
POST /api/auth/refresh/刷新 Access Token

✉️ 邮箱验证码

验证码规则：

  • 验证码长度：6位数字
  • 有效期：5分钟
  • 发送间隔：60秒
  • 验证码使用后立即失效

🔑 JWT Token

  • Access Token 有效期：60分钟
  • Refresh Token 有效期：7天
  • 认证方式：Bearer Token
  • Header: Authorization: Bearer {token}

👥 用户接口

GET /api/users/用户列表（仅管理员）
GET /api/users/me/当前用户信息
PATCH /api/users/me/更新当前用户
POST /api/users/change_password/修改密码

🛡️ 权限管理

  • normal - 普通用户：只能访问自己的信息
  • vip - VIP用户：可以查看 normal 和 vip 用户
  • admin - 管理员：可以管理所有用户

📝 请求示例

// 登录获取Token

POST /api/auth/login/

{ "email": "user@example.com", "password": "xxx" }


// 响应

{ "token": "eyJ...", "refresh": "eyJ...", "user": {...} }


// 访问受保护接口

GET /api/users/me/

Headers: Authorization: Bearer {token}
